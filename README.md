如何使用
===========

```bash 
$ groovy ILoveCk101.groovy [url]

```

for example

```bash
$ groovy ILoveCk101.groovy http://ck101.com/thread-2876990-1-1.html
$ groovy ILoveCk101.groovy http://ck101.com/beauty/
```
在 windows 桌面上就可以發現 ILoveCk101 資料夾，圖片都按照標題放在各自的資料夾下。
